# Sổ tay Phím tắt VS Code Thần thánh (Dành cho Mac)

Bí kíp giúp bạn code nhanh như chớp mà không cần dùng chuột.

## 1. Di chuyển con trỏ "thần tốc" (Điều hướng)
Thay vì bấm mũi tên từng chữ một, hãy dùng các tổ hợp sau:

- **`Option` + `←` / `→`**: Nhảy con trỏ theo **từng từ** (word by word) thay vì từng chữ cái. Cực kỳ tiện khi muốn lướt qua một tên biến dài.
- **`Cmd` + `←` / `→`**: Nhảy thẳng về **đầu dòng** hoặc **cuối dòng**. (Không cần dùng phím Home/End).
- **`Cmd` + `↑` / `↓`**: Nhảy thẳng lên **đầu file** hoặc **cuối file** code.

## 2. Bôi đen "nhanh như chớp" (Kết hợp phím Shift)
Chỉ cần thêm phím `Shift` vào các phím di chuyển ở trên, bạn sẽ bôi đen đoạn text đó:

- **`Shift` + `Option` + `←` / `→`**: Bôi đen từng từ một.
- **`Shift` + `Cmd` + `←` / `→`**: Bôi đen từ vị trí con trỏ ra đến hết dòng. (Rất hay dùng để xóa nhanh 1 dòng).

## 3. "Ma thuật" xoay vần dòng code (Rất hay xài)
- **`Option` + `↑` / `↓`**: Đổi chỗ dòng code hiện tại lên trên hoặc xuống dưới.
- **`Shift` + `Option` + `↑` / `↓`**: Nhân bản (Copy) dòng code hiện tại lên trên hoặc xuống dưới. Đỡ phải Cmd+C rồi Cmd+V!
- **`Cmd` + `Shift` + `K`**: Xóa bay màu nguyên dòng code hiện tại mà không cần bôi đen.

## 4. Sửa nhiều chỗ cùng lúc (Multi-cursor - Tính năng xịn nhất của VS Code)
- **`Cmd` + `D`**: Bôi đen một từ, sau đó cứ bấm `Cmd + D`, nó sẽ tự động tìm từ giống hệt tiếp theo và bôi đen cả 2, 3 từ. Bạn gõ 1 chữ thì cả 3 chỗ cùng thay đổi. Rất tiện để đổi tên biến!
- **Giữ phím `Option` + Click chuột**: Tạo ra nhiều con trỏ nhấp nháy ở nhiều chỗ khác nhau trên màn hình. Bạn gõ phím, chữ sẽ hiện ra ở tất cả những chỗ bạn vừa click.

## 5. Tìm kiếm và File
- **`Cmd` + `P`**: Mở thanh tìm kiếm file. Chỉ cần gõ tên file (ví dụ gõ "auth") rồi Enter là mở ngay file đó, không cần dùng chuột mò mẫm thư mục bên tay trái.
- **`Cmd` + `/`**: Comment (đóng khối) hoặc bỏ Comment dòng code hiện tại.

> **Bài tập thực hành:** Hãy mở một file code nháp lên, bỏ hẳn chuột ra và dùng phím mũi tên kết hợp `Option` hoặc `Cmd` để di chuyển, bôi đen, copy và di chuyển dòng code. Sau 3 ngày, bạn sẽ thấy mình code nhanh lên hẳn!
